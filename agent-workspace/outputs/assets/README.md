# assets/

Creative briefs and **links** to creative files. Keep big binaries (PSD, AI, raw video,
hi-res images) in cloud storage (ASUS DAM / Google Drive / OneDrive) and record the link
here — git is not for large media (see `.gitignore`).

Suggested:
- `creative-library.md` — index of available product shots, key visuals, logos, brand guidelines (with links)
- `briefs/` — design/video briefs
