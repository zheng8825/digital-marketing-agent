# campaigns/

One file (or sub-folder) per campaign. Suggested filename: `YYYY-MM_<slug>.md`
(e.g. `2026-06_back-to-school-vivobook.md`).

Each campaign doc should cover: objective & KPI, audience, key message/offer, channels &
budget split, creative needs, KOL list, UTM tags used, timeline, and a results/learnings
section filled in afterwards.
